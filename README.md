## Steps

1. Select clips as CLI args
1. Choose layout
1. Select clip
1. Start video, wait for keypress events, keypresses are recorded
    1. Start video
    1. Key for through gate
    1. Key for touched gate
    1. Key for missed gate
    1. Key for finished
1. Select clip again (allowed to redo completed)
1. Render video
